# MovieWebPractice

A lightweight **React + Vite** practice app for building a movie-style UI: cards, responsive grid, and API fetching. Perfect snapshot of my early front-end learning.

> Status: early practice project — polishing layout and basic interactions.

---

## ✨ Features

- Component-based UI with React  
- Responsive layout (cards/grid)  
- Simple state with Hooks  
- fetch from a public movie API (TMDB)  
- Fast local dev via Vite  

---

## 🧰 Tech Stack

- **Framework:** React  
- **Build Tool/Dev Server:** Vite  
- **Styling:** CSS (vanilla)  
- **Language:** JavaScript
